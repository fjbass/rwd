Main purpose

The purpose of this course is to introduce a set of theories and tools in order for students to obtain a proficient level of knowledge and gain a practical skillset for designing and developing responsive web sites for both PCs and mobile devices using basic web programming.

Having completed this course, students will have the knowledge to:
Describe the different file formats used in web development and their purpose. Create web sites using Hyper Text Markup Language (HTML5).
Reproduce webpage layouts using HTML5 and CSS3 when presented with images/screenshots of other websites. Use simple and advanced CSS3 selectors and properties to style webpages.
Select appropriate attributes for HTML5 elements. Apply the Bootstrap grid framework to create responsive websites.
Explain the difference between responsive and non-responsive websites. Utilize the Bootstrap classes to apply styling to responsive websites.
Test HTML5 files for errors using the W3C markup validator. Implement JavaScript functions to add functionality to websites.
Account for the difference between the JavaScript and Java programming languages. Use XMLHttpRequest to read content from an external source and integrate this content into a website.
  Select HTML elements and apply jQuery animations to the selected elements to make websites interactive.
